# Digitalisasi Pesantren

## Fasilitas Website Profil Pesantren

Layanan digitalisasi pesantren menyediakan fasilitas website profil pesantren yang dapat digunakan oleh setiap pesantren untuk menampilkan informasi berikut:

- Profil lembaga (sejarah, visi misi, struktur organisasi)
- Informasi program pendidikan dan kegiatan
- Galeri foto dan video
- Kontak dan lokasi
- Berita dan pengumuman
- Link pendaftaran santri baru (jika tersedia)
- **Asisten AI Terintegrasi (Pisantri AI)** untuk layanan informasi mandiri
- Integrasi dengan sistem administrasi dan aplikasi santri

Website profil ini dapat diakses melalui subdomain khusus setiap pesantren, misal: `namapesantren.netlify.app` atau domain kustom sesuai kebutuhan.

Fasilitas ini memudahkan pesantren untuk membangun citra digital, memperluas jangkauan informasi, dan meningkatkan layanan kepada santri, wali santri, serta masyarakat umum.

> Untuk aktivasi website profil, silakan hubungi admin atau gunakan fitur pendaftaran tenant baru pada sistem.
